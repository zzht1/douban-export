如何获取豆瓣 Cookie

只需 3 步，大约 1 分钟：

1登录豆瓣，打开开发者工具

打开 任意豆瓣条目页，按 F12 键，点击顶部的 Network 标签。

2刷新页面，找到请求

点击左上角 ⊘ 清除旧记录，然后按 F5 刷新。在请求列表中找到包含你 ID 的那条（如 4908885/），点进去。

3复制 Cookie

在右侧 Request Headers 中找到 cookie: 那一行，复制其中 dbcl2="xxx:xxx" 的值（不含分号）。或者也可以复制整行 cookie 字符串。
